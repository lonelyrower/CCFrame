# 移动端 PWA 指南

本指南汇总了在手机或平板上安装 CCFrame 并获得最佳触控体验的建议。

## 安装步骤

1. 确保站点已通过 HTTPS 部署并启用了 `app/manifest.ts`、`app/sw.ts`（默认模板已配置）。
2. 在移动端浏览器打开站点：
   - **iOS Safari**：点击分享按钮 → 选择 “添加到主屏幕”。
   - **Android Chrome/Edge**：地址栏下方或右上角菜单会提示 “安装应用”。
3. 安装完成后从主屏幕启动，即可以原生应用形式运行（独立窗口、无浏览器地址栏）。

## 触控手势与操作

- **双指捏合**：在光箱中缩放照片，放大缩小时会保持手指中点位置。
- **双指拖动**：快速切换到上一张/下一张照片。
- **单指拖动**：放大状态下，可在图像范围内平移浏览；边界自动限制，避免 “丢失” 图片。
- **双击 / 双指轻点**：在当前触点位置快速切换 100% ↔︎ 200% 缩放。
- **触控提示**：Lightbox 帮助面板（快捷键按钮）新增 “Touch & Trackpad” 区域，可随时回顾手势说明。

## 性能建议

- 为获得最佳离线体验，请确保：
  - `app/sw.ts` 中的缓存策略未被禁用。
  - 在部署端通过 `npm run build && npm run start` 的生产模式运行。
- 移动端首屏建议开启图片渐进式加载：当前 `PhotoZoomCanvas` 会自动使用 blurhash / dominant color 做过渡。
- 如果照片较大，推荐在后台执行 `npm run variants:backfill` 生成更多 WebP/AVIF 变体，以减少网络带宽。

## 环境健康检查

发布前可使用下列命令验证依赖服务是否齐备：

```bash
npm run healthcheck -- --strict
```

缺失的 Redis / 数据库会以 `WARN` 或 `ERROR` 呈现，建议在 PWA 部署环境中确保这些服务可用，以获得完整的缓存与队列能力。
