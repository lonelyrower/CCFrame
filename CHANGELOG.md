# CCFrame 变更日志

## [2025-09-17] - AI功能模块移除

### 🔴 移除的功能
- ❌ **所有AI相关功能** - 根据用户需求，完全移除AI相关功能
  - 删除 AI增强、AI生成、背景移除等功能
  - 删除 语义搜索和embedding功能
  - 移除 `app/admin/ai-*` 和 `app/admin/semantic` 相关页面
  - 移除 `app/api/ai/*` 和 `app/api/search/semantic` 相关API
  - 移除 `components/ai/*` 和语义搜索组件
  - 移除 `lib/ai-*.ts`、`lib/embeddings.ts`、`lib/semantic-*.ts` 等相关库文件

### 🟢 保留的功能
- ✅ **图片处理管线** - 保留基础图片处理 (Sharp + 多格式输出)
- ✅ **标签系统** - 保留手动标签管理功能
- ✅ **基础搜索** - 基于标签和EXIF信息的搜索

### 配置变更
- 更新 `.env.example` - 移除所有AI相关API密钥配置
- 更新 `.env.production.example` - 移除AI处理和语义搜索配置
- 更新 `lib/constants.ts` - 移除AI相关作业类型

### 文档更新
- 更新 `README.md` - 重新描述功能特色，专注照片管理
- 更新 `DEPLOYMENT_STATUS.md` - 记录AI功能移除状态
- 更新路线图 - 调整功能规划方向

### 用户影响
- 用户将无法再使用AI图片增强、生成、背景移除、语义搜索等功能
- 专注于传统的照片管理和基于标签的搜索功能
- 照片管理、标签、相册等核心功能正常使用

### 技术债务清理
- 移除未使用的AI和语义搜索相关依赖配置
- 清理相关的TypeScript类型定义
- 简化后台任务队列类型
